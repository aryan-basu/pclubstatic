About Us

Programming Club UIET

In this age of increasing automation, programming is not limited to only computer science and related fields. In order to survive in the competitive world, every competent engineer needs to know how to program, making the Programming Club one of the most important clubs in the institute. Programming Club (or better known as Pclub) understands this responsibility and plays a crucial role in sustaining the coding culture in the institute. With various introductory workshops and informative seminars, we introduced the students to the diverse facets of programming. We organised regular competitions for the general student body to test and sharpen their programming skills. Pclub also has an essential role in the successful conduction of SFD, the institute’s techno-management festival. Hacktoberfest, an event organised by the Pclub, attracts the maximum participation in the fest. While the club has been flourishing well over the years, it still has a lot to conquer. Some of the noteworthy initiatives taken by our current secretaries include the formation of smaller groups focusing on specific areas (competitive programming group, development group and machine learning group) and the fresher’s mentorship program. Still, the club has a lot to improve upon to take this momentum forward and make a name in the programming world.

Projects

  • hacktoberfest

    Hacktoberfest is a month-long celebration of open-source software, held every year in the month of October. During this event, everyone can support open-source coding by contributing changes to the project, earning a giveaway worth ₹1000 as a reward. Regardless of their background and experience, we invite everyone to participate and contribute to the numerous Pclub projects. One of the few ways to contribute to a project is by working on the code and documentation, creating new artwork and contributing to localization. You can also share your experiences with Pclub by creating a new blog post. Generally, all the pull requests in GitHub would qualify and help you become a part of the open source community.

  • Software freedom Day

    Software Freedom Day is a worldwide event to celebrate the usage of Free and Open Source Software. It is a public education effort to increase awareness of free software and its virtues and encouraging its use. Each year the event has seen a growing number of participants. Our past speakers were from many prestigious software companies like Facebook, Microsoft and Google, enlightening us about the various software developments. SFD aims to bring together developers and students, of all levels, on a single platform to share their thoughts, ideas and inspirations. This year, the Programming Club of UIET brings you the 11th Edition of Software Freedom Day on September 19, 2020, and September 20, 2020.

    </> Source code

  • PSoC

    Pclub Summer of Code is the two-month-long Open Source program conducted during summers by Programming Club, UIET. It aims to introduce beginners to Open Source Development while encouraging diversity. Throughout the program, participants contribute to different projects under the guidance of experienced mentors. Mentors are able to bring their ideas to life while the Mentees learn about the tech. PSoC is primarily for the people who want to delve into Open Source Development or are already into FOSS development. This program helps students gain experience for other open-source programs like GSoC, Outreachy, etc.

    </> Source code

  • Covid-19 Detector

    Covid-19 Detector was a PSoC project of Sayed and his team. This project notifies the user about nearby covid cases through the data pulled in from various sources, verified by cross-referencing with official data like WHO. It helps the user find a nearby covid test centre while keeping them up to date with the latest initiatives taken by WHO and the government over Covid-19. The app fetches data from the Aarogya Setu app, so the user's quarantine status automatically gets updated. Also, any information from this app can be shared via any sharing medium.

    </> Source code

  • Androiders Buddy

    Androiders Buddy was a PSoC project of VBSquad. This project helps the mentees become familiar with many of the Android specificities and prepares them for more advanced coursework in Android development. Hence this beginner-friendly project aimed to make new learners in Android Development get acquainted with the fundamental topics/concepts of Android Studio. Androiders buddy is a repository that contains projects for all the subjects of android.

    </> Source code

  • Pocket News

    Pocket news was a PSoC project of UIET students Ankay and his team. It's a cross-platform app made with the flutter framework that gives you the top 30 news feeds with just one tap of your finger. It is backed with Firebase authentication services so that you could register yourself and log in to get all top international and national headlines in just less than a second. With a multilingual news category, you can also share any news article via any sharing medium.

    </> Source code
